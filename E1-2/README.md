# 콘솔 퀴즈 게임

파이썬 기본 문법만 사용해 구현한 콘솔 기반 축구 퀴즈 게임입니다.  
사용자는 메뉴를 통해 축구 퀴즈를 풀고, 새 퀴즈를 추가하거나 삭제하고, 전체 퀴즈 목록과 최고 점수, 점수 기록을 확인할 수 있습니다.

## 프로젝트 개요

- 콘솔 메뉴 기반 퀴즈 게임
- `Quiz`, `QuizGame` 두 개 이상의 클래스로 구성
- 퀴즈 데이터, 최고 점수, 게임 기록을 프로젝트 루트의 `state.json`에 UTF-8로 저장
- 프로그램을 다시 실행해도 추가한 퀴즈, 최고 점수, 게임 기록이 유지됨
- 데이터 파일이 없거나 손상되어도 기본 퀴즈로 복구 가능

## 퀴즈 주제와 선정 이유

기본 퀴즈 주제는 **축구 상식**입니다.  
누구나 비교적 쉽게 접근할 수 있으면서도 문제를 다양하게 만들 수 있어 기본 주제로 선정했습니다.

## 실행 방법

```bash
python main.py
```

Python 3.10 이상을 권장합니다.

## 기능 목록

- 퀴즈 풀기
- 퀴즈 출제 순서 랜덤 섞기
- 퀴즈 풀기 전 문제 수 선택
- 풀이 중 힌트 보기
- 힌트 사용 시 해당 문제 점수 미획득 처리
- 퀴즈 추가
- 퀴즈 삭제
- 퀴즈 목록 보기
- 최고 점수 확인
- 점수 기록 보기
- 파일 저장/불러오기
- 잘못된 입력, 빈 입력, 범위 밖 숫자 처리
- `Ctrl+C`, `EOF` 발생 시 저장 후 안전 종료

## 데이터 파일 설명

프로젝트 루트의 `state.json` 파일을 사용합니다.

### 역할

- 퀴즈 목록 저장
- 최고 점수 저장
- 게임 기록 저장

### 스키마 예시

```json
{
  "quizzes": [
    {
      "question": "문제 내용",
      "choices": ["선택지1", "선택지2", "선택지3", "선택지4"],
      "answer": 2,
      "hint": "문제 풀이에 도움이 되는 힌트"
    }
  ],
  "best_score": 5,
  "history": [
    {
      "played_at": "2026-04-02 08:00:16",
      "question_count": 3,
      "score": 2
    }
  ]
}
```

## 메뉴 구성

```text
1. 퀴즈 풀기
2. 퀴즈 추가
3. 퀴즈 삭제
4. 퀴즈 목록 보기
5. 최고 점수 확인
6. 점수 기록 보기
7. 종료
```

## 예외 처리 정책

- 숫자 입력 시 앞뒤 공백 제거 후 처리
- 숫자 변환 실패 시 재입력 유도
- 허용 범위 밖 숫자 입력 시 재입력 유도
- 빈 입력 시 재입력 유도
- 퀴즈 풀이 중 `1~4`, `h` 외 입력 시 재입력 유도
- `KeyboardInterrupt`, `EOFError` 발생 시 가능한 범위에서 저장 후 종료
- `state.json`이 없으면 기본 퀴즈 데이터로 시작
- `state.json`이 손상되면 안내 메시지 출력 후 기본 퀴즈로 복구
